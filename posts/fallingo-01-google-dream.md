# 나의 꿈은 구글이니까

2025년 5월 말, 폴링고(fallingo) 앱을 만들기로 결심하고 가장 먼저 한 결정이 있었다.

> "나의 꿈. 하지만, 갈 수 없이 너무 멀리 있기에 내가 만드는 앱의 모든 기술스택을 구글로 결정했다."

감상적인 결정이었다.

## 왜 구글인가

어릴 적부터 구글은 '도달할 수 없는 꿈'이었다.

실리콘밸리, 세계 최고의 기술 기업, 혁신의 아이콘... 거창한 말이 아니라 그냥 단순하게 **"여기서 일하고 싶다"**는 마음.

하지만 현실은 냉정하다. 나이도 있고, 한국에 있고, 영어도 완벽하지 않다. 갈 수 없이 너무 멀리 있는 곳.

그래서 생각했다.

> "갈 수는 없지만, 구글의 기술로 내 앱을 만들면 되지 않을까?"

## 100% 구글 기술 스택

그래서 정했다. **모든 기술을 구글로.**

**백엔드:**<br />
- FastAPI + Python 3.11+<br />
- PostgreSQL 15 + PostGIS<br />
- Cloud Run 배포

**프론트엔드:**<br />
- Flutter 3.24+<br />
- Firebase Auth

**AI 서비스:**<br />
- Vision API (이미지 분석)<br />
- Gemini (AI 대화)<br />
- Document AI (문서 처리)<br />
- Translate API (번역)

**인프라:**<br />
- Cloud Run (서버리스 컨테이너)<br />
- Cloud Storage (파일 저장)<br />
- Cloud SQL (데이터베이스)<br />
- Memorystore Redis (캐싱)

**지도:**<br />
- Google Maps Platform

백엔드도 모르고, Flutter도 처음이고, 구글 클라우드는 더더욱 몰랐다.

하지만 **AI와 함께라면 된다고 믿었다.**

## 갈 수 없는 꿈

구글이라는 회사에 들어갈 순 없다.

나이도 있고, 한국에 있고, 영어도 완벽하지 않다.

하지만 구글의 기술로 내 앱을 만들 수는 있다.

---

모르는 게 산더미였다. FastAPI도, Flutter도, Cloud Run도 전부 처음이었다.

하지만 걸음은 이미 시작됐다.

> "AI와 함께라면, 내가 모르는 기술도 배우면서 만들 수 있지 않을까?"

그리고 실제로 그렇게 됐다.

**나의 꿈은 구글이니까.**
