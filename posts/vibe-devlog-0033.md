# vibe 개발일지 #33 - 성능 최적화와 피그마 엔진의 도약 (12개 커밋)

**작업 기간**: 2026-04-08 ~ 2026-04-09

안녕하세요, vibe를 개발하고 있는 Su입니다. 

지난 이틀은 그야말로 '폭풍 같은 코딩'의 시간이었습니다. 단순히 기능을 추가하는 것을 넘어, AI가 만들어내는 결과물의 품질을 어떻게 하면 '최소한의 가이드(Harness)'로 '최대한의 품질'까지 끌어올릴 수 있을지 고민이 깊었거든요. 그 결과물들을 모아 드디어 **v2.9.0** 버전을 배포했습니다.

이번 업데이트의 핵심은 **"더 빠르고, 더 정확하게"**입니다. 12개의 커밋에 담긴 변화들을 정리해 보았습니다.

---

## 📝 이번 기간 작업 내용

### 1. 코어 성능 및 아키텍처 최적화 (P1, P2, P3 적용)
vibe의 엔진에 해당하는 실행 로직을 세 단계(P1~P3)로 나누어 대대적으로 개편했습니다.
- **P1(성능):** 동시성 파티셔닝(Concurrency Partitioning)을 도입하고, 에러 발생 시 즉시 중단하는 대신 보류(Pending) 처리하여 복구 기회를 주는 로직을 추가했습니다. 스트리밍이 겹치는 구간의 오버헤드도 대폭 줄였습니다.
- **P2(안정성):** **코디네이터(Coordinator) 패턴**을 도입했습니다. 이제 훅(Hook) 시스템이 입력을 검증하고, 불필요하게 비대해진 컨텍스트는 자동으로 압축 트리거를 작동시킵니다.
- **P3(운영):** `postinstall` 구조를 정리하여 설치 경험을 개선했고, 개발자가 AI 사용 비용을 실시간으로 추적할 수 있는 누적 조회 유틸리티를 추가했습니다.

### 2. Figma to CSS 추출 품질 고도화
AI가 디자인을 보고 코드를 짤 때 가장 많이 실수하는 부분들을 '스킬' 차원에서 해결했습니다.
- **추출 엔진 보강:** `extractCSS` 로직을 개선하여 Figma 속성 추출 품질 목표를 **90%**까지 끌어올렸습니다.
- **누락 속성 대응:** 현업에서 필수적인 `individual stroke`, `dash`, `mask`, `text-case` 등 4종의 속성 누락을 해결했습니다.
- **이미지 판별 고도화:** 배경 이미지나 카드 컴포넌트를 통째로 이미지로 인식해버리는 고질적인 문제를 해결하기 위해 `BLOCKING` 판별 로직과 실패 사례 명시 규칙을 도입했습니다.

### 3. 문서화 및 DX(개발자 경험) 개선
- **README 리뉴얼:** 사용자가 `/vibe.spec` 작성부터 `/vibe.run` 실행까지의 워크플로우를 한눈에 이해할 수 있도록 플로우 중심으로 개편했습니다. 상세 내용은 Wiki로 분리하여 접근성을 높였습니다.
- **스킬 문서 정규화:** 3개 파일에 흩어져 중복되던 스킬 설명들을 역할별로 분리하고 정리하여 AI가 혼동하지 않도록 리팩터링했습니다.

**숫자로 보는 작업량:**
- **v2.8.53 → v2.9.0** 메이저에 준하는 기능 업데이트
- **4종**의 핵심 CSS 속성 정밀 대응
- **3단계**의 성능 최적화 파이프라인 구축

## 💡 작업 하이라이트

### "코디네이터 패턴과 훅 검증"
이번 작업 중 가장 공을 들인 부분은 **코디네이터 패턴**의 도입입니다. 이전에는 AI 에이전트가 훅을 호출할 때 입력값이 유효한지 체크하는 로직이 파편화되어 있었습니다. 

이제는 중앙 코디네이터가 훅의 입력을 먼저 검증하고, 실행 전후의 상태를 관리합니다. 이로 인해 "잘못된 인자로 인한 런타임 에러"가 획기적으로 줄었습니다. AI에게 "이건 틀렸어, 다시 해"라고 말하는 대신, 하네스(Harness)가 "이런 형식으로만 들어올 수 있어"라고 길을 닦아주는 셈이죠.

### "Figma 속성 추출 품질 90% 도전"
디자인을 코드로 옮길 때 AI가 `stroke-dasharray`나 `text-transform` 같은 세밀한 속성을 놓치는 경우가 많았습니다. 8일 커밋(5120c58)을 통해 이 부분의 추출 로직을 보강했습니다. 이제 복잡한 벡터 그래픽이나 타이포그래피 설정도 꽤나 정확하게 CSS로 변환됩니다. FE 개발자로서 가장 뿌듯한 순간이었습니다.
