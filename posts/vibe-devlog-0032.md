## 💡 작업 하이라이트

**이미지 추출 png → webp 전환**

Figma에서 추출하는 이미지 포맷을 PNG에서 WebP로 전환했습니다. 동일 품질에서 파일 크기가 25~35% 줄어듭니다. 동시에 이미지와 HTML의 판별 규칙도 강화하여 텍스트가 이미지로 추출되는 문제를 방지했습니다.

**하네스 자기 관리 (Self-Management)**

하네스가 자기 자신의 상태를 관리하는 기능을 추가했습니다:

```
세션 시작
    ↓
버전 체크: npm 글로벌에 설치된 vibe 버전 확인
    ↓
업데이트 필요? → 안내 메시지 표시
    ↓
세션 중:
  - LLM 비용 추적 (토큰 사용량 모니터링)
  - 롤백 체크포인트 (위험한 작업 전 자동 저장)
  - 에스컬레이션 (3번 실패 시 사용자에게 알림)
```

**Agent = Model + Harness 정의**

README와 문서에서 하네스를 업계 표준 용어로 재정의했습니다:

- **Agent** = Model + Harness
- **Model**: Claude, GPT, Gemini 등 LLM
- **Harness**: vibe가 제공하는 스킬, 훅, 에이전트, 설정의 총체

vibe는 Model이 아니라 **Harness**입니다. 어떤 LLM과도 결합 가능합니다.

v2.7.1에서 시작한 이 시리즈가 v2.8.51에 도달했습니다. 272개의 커밋, 15편의 개발일지(#18~#32)로 기록된 약 2개월간의 여정입니다.
